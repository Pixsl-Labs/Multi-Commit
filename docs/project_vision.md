Multi-Commit Vision

Purpose:
A Linux desktop productivity tool that acts as a project cockpit.

Primary Features:
- Git workflow
- Project management
- Commands
- Notes
- Checklists
- Roadmaps

Target Users:
- Developers
- Students
- Linux users
- Indie hackers

Core Principle:
Reduce friction between coding and project management.

Non-Goals:
- Full Jira replacement
- Full GitHub replacement
- Enterprise project management
